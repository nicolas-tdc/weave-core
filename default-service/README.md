# weave-service

**From service's root directory**

# Setup

```bash
cp .env.dist .env
```

# Commands

- Start service
```bash
./weave.sh start
```

- Stop service
```bash
./weave.sh stop
```

- Log service's available ports
```bash
./weave.sh log
```

- Update service
```bash
./weave.sh update
```

- Execute service's backup-task
```bash
./weave.sh backup-task```
