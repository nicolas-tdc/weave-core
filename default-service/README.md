# weave-service

# Setup

- Copy environment file
**From service's root directory**
```bash
cp .env.dist .env
```

- Set environment variables: edit copied .env file

# Commands
*From service's root directory*

- Start service
```bash
./weave.sh start
```

- Stop service
```bash
./weave.sh stop
```

- Log service's available ports
```bash
./weave.sh log
```

- Update service
```bash
./weave.sh update
```

- Execute service's backup-task
```bash
./weave.sh backup-task```
